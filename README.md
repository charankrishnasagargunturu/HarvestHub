# 🌾 Harvest Hub

A smart farming platform connecting India's farmers with real-time crop prices, market insights, and farming tips.

## Features

- 🔍 **Crop Price Search** – Check crop prices across all Indian states
- 🔔 **Alerts & Tips** – Emergency alerts and farming best practices
- 👤 **User Profile** – Manage your farmer profile
- 🔐 **Authentication** – Secure login, sign-up with OTP-based forgot password

## Tech Stack

- **Mobile App**: Flutter / Dart (see harvest hub code.txt)
- **Web App**: Pure HTML5, CSS3, Vanilla JS (see index.html)
